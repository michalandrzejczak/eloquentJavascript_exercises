Deep comparison

Compare 2 objects using deepEqual(object1, object2)
For example:
var A = {value: 10, rest: {value: 20, rest: null}};
var B = {value: 10, rest: null};
console.log(deepEqual(A,B));
expected: false
B = {value: 10, rest: {value: 20, rest: null}};
console.log(deepEqual(A,B));
expected: true
